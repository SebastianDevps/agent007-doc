## [5.1.1] — 2026-03-29 — "ECC Integration (P0 + P1)"

### Added

**GAP-007: Pre-commit quality hook** — `hooks/pre-commit-guard.py` (PreToolUse/Bash) intercepts `git commit` before execution:
- Blocks staged `.env` files
- Blocks hardcoded secrets patterns (`API_KEY=`, `SECRET=`, `PASSWORD=`, `TOKEN=` with literal values)
- Blocks `debug` statements (e.g. the JS `debugger` keyword)
- Warns on `console.log/debug/warn`
- Validates Agent007 pipe-delimited commit message format

Paired with `hooks/block-no-verify.py` which blocks all `git commit --no-verify` calls. Both registered in `settings.json` PreToolUse/Bash chain.

## [5.1.0] — 2026-03-29 — "Context Freedom"

### Added

- **Fresh subagents per task** — each task spawns an isolated subagent with a clean context window; eliminates cross-task context bleed
- **Wave scheduler** — complex tasks grouped into dependency waves; waves execute in parallel via the Agent tool, reducing total wall-clock time
- **Context budget auto-tracking** — `context-window-guard.py` reads actual token usage; emits `⚠️ context budget` warning at 80%, blocks generation at 95%
- **RTK (Resume Token Key) integration** — `agent007-init.js` reads `.sdlc/state/session.md` on every SessionStart; active task triggers resume banner automatically
- **Continuous learning / Instinct system** — `/instinct-status` and `/evolve` allow observed patterns to be promoted to permanent skills
- **Discuss-lock phase** — `plan.md` enforces DECISIONS.md gate before execution; cannot bypass without `--force`
- **Security scanning** — `/security-scan` backed by `scripts/security-scan.js`; Grade A (0 critical, 0 warnings) achieved
- **Config protection** — `hooks/config-guard.py` blocks writes to linter/TypeScript configs during code generation

### Changed

- Pipeline skill aliases removed (4 files) — all callers use canonical names: `Skill('brainstorming')`, `Skill('finishing-a-development-branch')`, `Skill('subagent-driven-development')`, `Skill('using-git-worktrees')`
- Internal command redirects cleaned (8 stubs removed from `commands/`)
- Skill counts corrected: `25 active skills` (19 invokable + 6 auto-injected)

### Removed

- 4 pipeline alias stubs
